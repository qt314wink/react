## 🚀 What This Is

This is a personal React starter scaffold used as the foundation for UI experiments, design system demos, and creative tool prototypes. It's intentionally lean — no framework opinions baked in beyond React itself — so it can be adapted quickly for shader galleries, generative art components, or design system showcases.

---

## 📦 Stack

- **React 18+** with hooks
- **TypeScript** for type safety
- **Vite** for fast dev builds
- **CSS Modules** or Tailwind (per project)
- Deployable to **Vercel** or **GitHub Pages**

---

## 🧩 Intended Uses

- Bootstrap new design system component demos
- Shader / WebGL canvas wrapper experiments
- Quick generative art tool prototypes
- UI component playgrounds
